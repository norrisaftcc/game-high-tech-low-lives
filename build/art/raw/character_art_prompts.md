## TECHNICAL NOTES FOR ALL IMAGES

**Ink Line Art Style:**
- Variable line weight for depth
- Clean technical illustration base
- Duotone applied as overlay washes, not replacing linework
- High contrast between lit and shadow areas
- Readable detail in both color zones

**Duotone Application:**
- Primary color: Highlights, key features, chrome/tech
- Secondary color: Shadows, ambient, depth
- Limited third color only where specified (gold accents for Kaida)

**Consistency Across Set:**
- All use same ink line art approach
- Portrait compositions all head-and-chest, similar framing
- Schematics all use AR HUD/technical display format
- Each character's duotone is distinctive and consistent

**Portrait vs Schematic Contrast:**
- Portraits: More organic, human, personality visible
- Schematics: Cold technical documentation, surveillance perspective
- Together they show: How the character sees themselves vs. how systems see them
