## CHROME (Choose 3—Yes, three)

Hackers are more chrome than flesh. Each piece shifts your stress tracks dramatically.

### ☐ Neural Interface Suite (REQUIRED)
*Direct brain-to-net connection. You don't use computers, you become them.*
- **Effect**: +2 to Code when jacked in directly
- **Stress Shift**: +3 Systems, -2 Meat (neural load, body neglect)
- **Maintenance**: 800 credits/month
- **When it breaks**: Can't jack in directly. Code at -3 until repaired. Withdrawal symptoms.

### ☐ Cognitive Processor Enhancement
*Overclocked wetware. You think faster than baseline humans.*
- **Effect**: +1 to Code for complex decryption, multi-system management
- **Stress Shift**: +2 Systems, -1 Nerves (information overload, emotional flatness)
- **Maintenance**: 600 credits/month
- **When it breaks**: Thought loop errors, memory corruption. Take 1 Nerves stress per scene until fixed.

### ☐ Data Visualization Matrix
*See code as architecture, networks as physical spaces.*
- **Effect**: +2 to Code when navigating complex networks or finding hidden data
- **Stress Shift**: +2 Systems, -1 Meat (visual processing strain)
- **Maintenance**: 500 credits/month
- **When it breaks**: Reality and cyberspace blur. Compel this for hallucinations, navigation errors.

### ☐ Black ICE Countermeasures
*Experimental defensive chrome. Fights back against lethal security.*
- **Effect**: When targeted by Black ICE, roll Code to counter. Success: negates. Failure: half damage
- **Stress Shift**: +2 Systems, -1 Nerves (paranoia, threat assessment dysfunction)
- **Maintenance**: 700 credits/month
- **When it breaks**: Next Black ICE attack hits at double strength. You don't realize it's broken until too late.

### ☐ Netrunner's Rig (External Gear, not installed chrome)
*Portable deck, custom built, more powerful than standard hardware.*
- **Effect**: +1 to Code for all netrunning when using your rig. Required for deep dives.
- **Stress Shift**: +1 Systems (represents carrying external hardware stress track)
- **Maintenance**: 400 credits/month (hardware upkeep)
- **When it breaks**: Can still hack, but at -2. Repair takes time and rare components.

### ☐ Memory Buffer Array
*External memory storage. Perfect recall of any data stream you've seen.*
- **Effect**: Once per session, recall any piece of data perfectly. Review network layouts, access codes, file contents
- **Stress Shift**: +1 Systems, -1 Nerves (disconnection from organic memory)
- **Maintenance**: 450 credits/month
- **When it breaks**: Memories feel corrupted. Can't trust your own recall. -1 to Cool.

## TALENTS (Choose 2)

### ☐ Deep Dive Specialist (4 points)
When doing extended netrunning contests, you can absorb one additional exchange before consequences. Once per session, automatically find a backdoor or exploit in any system you study for one minute—no roll required. You see the vulnerability.

### ☐ Ghost Protocol (2 points)
+2 to Code when covering your tracks, hiding your identity, or disappearing from networks. You're very good at not being found.

### ☐ Multi-System Management (2 points)
You can maintain connections to multiple systems simultaneously. Run two netrunning actions per exchange in extended contests (but you still only defend once—you're exposed).

### ☐ Exploit Library (2 points)
You maintain an updated database of zero-days and exploits. +2 to Code when attacking systems you've researched (define "researched" as studying known system architecture or encountering similar security before).

### ☐ Meat Puppet Override (2 points)
Once per session, hack into someone's personal chrome (reflex boosters, HUD, combat systems) and shut it down. They roll to resist with their Code (not their chrome stat). Success: their chrome stops working for one exchange.

### ☐ Remote Operations (2 points)
You can participate in crew operations while physically elsewhere (your apartment, a safehouse, the back of the van). As long as you have a connection, you're there digitally. Vulnerable if someone finds your meat body.

## HOW TO PLAY THE HACKER

### YOU'RE GOOD AT:
- Deep dives into corporate networks
- Breaking ICE, bypassing security
- Data theft, surveillance, electronic warfare
- Multi-tasking across systems
- Finding exploits no one else sees
- Staying anonymous
- Working remotely while crew operates physically

### YOU'RE BAD AT:
- Physical confrontation (you're fragile)
- Social interaction (people aren't systems)
- Staying present in meatspace
- Taking care of your body
- Recognizing emotional cues
- Walking away from a mystery
- Disconnecting when you should

### WHEN YOU ROLL:
- **Body**: You probably don't (and fail when you do)
- **Reflexes**: Unplugging fast, dodging while jacked in (rare)
- **Code**: This is everything—hacking, breaking ICE, cyber warfare, data theft, navigating networks
- **Tech**: Hardware repair, understanding physical systems, jury-rigging connections
- **Cool**: Staying calm under digital assault, managing multiple feeds, cold reading people through their data trail

## THE HACKER MOVE: GHOST IN THE MACHINE

Once per session, when you're jacked into a system, you can become truly untraceable:

**"I'm a ghost in the machine."**

For the rest of the scene, you operate as pure information—you cannot be found, traced, kicked out, or hurt by ICE. You're invisible to security. You can do one thing with complete impunity:
- Steal critical data
- Plant false information
- Sabotage a system
- Open every door in a facility
- Shut down security for exactly 90 seconds

Then you're gone. No trace you were ever there.

**Cost**: This move crashes your neural interface afterward. Take Moderate Consequence: *Chrome Burnout* and you can't hack for 24 hours while your wetware recovers.
